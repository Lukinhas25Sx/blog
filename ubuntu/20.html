Ubuntu 20.04 LTS (Focal Fossa)


Ubuntu 20.04 LTS (Focal Fossa):
Lançada em abril de 2020.
Outra versão LTS com suporte de longo prazo.
A transição do ambiente de desktop Unity para o GNOME como interface padrão foi concluída.
Apresentou o GNOME 3.36, que trouxe melhorias de desempenho e novos recursos para o ambiente de desktop.
Introduziu o suporte aprimorado para o sistema de arquivos ZFS, que oferece recursos avançados de gerenciamento e proteção de dados.
Melhorias no suporte a dispositivos de Internet das Coisas (IoT), incluindo integração com o aplicativo Ubuntu Core.
