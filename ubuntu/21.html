Ubuntu 21.04 (Hirsute Hippo)


Ubuntu 21.04 (Hirsute Hippo): Lançada em abril de 2021. Uma versão regular, não LTS. Continuou com o GNOME 3.38, oferecendo melhorias adicionais no desempenho, interface do usuário e recursos. Introduziu o suporte inicial para o Wayland como servidor gráfico padrão, embora o X.org ainda estivesse disponível como opção. Incluiu atualizações de pacotes, correções de bugs e aprimoramentos em diversos aspectos do sistema operacional. Foi uma versão de curta duração, pois o Ubuntu 21.10 foi lançado apenas seis meses depois.
